## How to Run

1. **Set up the virtual environment:**
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # macOS/Linux:
   source venv/bin/activate
   ```

2. **Install required packages:**
   ```bash
   pip install numpy pandas seaborn matplotlib scikit-learn ipykernel nbformat nbconvert
   ```

3. **Navigate to the target task folder and run the scripts or notebook:**
   - For Jupyter Notebook (Task 06):
     ```bash
     jupyter notebook Task06-Exploratory-Data-Analysis/eda_stroke_prediction.ipynb
     ```
